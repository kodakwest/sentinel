Create rebranding concepts and a visual mood board for Sentinel, a clinical pharmacology assistant for registered nurses. The current working name is "Sentinel" but the personality that users love is "Nurse Clippy" — the AI nurse assistant persona.

ABOUT:
- Cloudflare Workers app (TypeScript, React frontend) that gives nurses drug info, FDA label data, clinical pharmacology answers
- Core personality: Nurse Clippy — direct, clinically precise, no fluff. Speaks like an experienced nurse educator who's seen it all
- Users are RNs, nursing students, clinical educators — needs to feel trustworthy and clinical, not consumer-health
- Medical reference tool first, AI assistant second

NAMING DIRECTION:
- Core concept: Keep "Nurse Clippy" as the brand personality/assistant name. The platform/product name can be something else (like how "Clippy" was the assistant for Microsoft Office)
- Tone: Clinical, trustworthy, warm-but-professional. Not sterile or cold like a hospital, not casual like a consumer app
- Medical/nursing feel — think stethoscope, clipboard, scrubs, clinical confidence
- Words to explore: clippy, nurse, clinical, rx, dose, chart, stat, pdx (pharmacology data exchange), formulary, vade (vade mecum), compendium
- Words to avoid: sentinel (current name, may keep or replace), anything too military/security sounding

PRODUCT ECOSYSTEM:
- The AI persona is "Nurse Clippy" — this is the face of the product
- The platform/product name is what goes on the website/header — could be anything
- Future: potential mobile companion for floor nurses, clinical decision support

REQUIREMENTS:
1. Present 5 name candidates for the PRODUCT (not the assistant) with detailed rationale for each — what it means, how it feels for nurses, why it works
2. Keep "Nurse Clippy" as the assistant persona name throughout
3. Create a dark-themed HTML mood board with:
   - Color palette (primary, accent, supporting colors with hex codes) — coral/rose medical accent
   - Typography pairings (headings + body) — clean and clinical
   - 2-3 SVG logo concept sketches — mark should feel medical but warm, think plus/health cross, clipboard, RX symbol, or an abstract mark that combines care + pharmacology
   - Tagline options for the top 2-3 name candidates
   - Overall visual direction — textured, clinical-but-warm, trustworthy
4. Save as `docs/rebrand-moodboard.html` in the repo (in docs/ folder, not root)

Design conventions: dark theme (#0e0f0d bg), coral/rose accent (#e07d5f, #fb7185), Inter font, clean modern lines. The output should be a single self-contained HTML file with all CSS/SVG inline.
