Sentinel — Boris Data Flow

How drug data moves from external sources to nurse-friendly UI

Drug Pipeline

  • • Search triggers assembleDrug from DailyMed
  • • Dossier load fires enrichDrugFromFda (async)
  • • AI remap creates nurse-friendly summaries
  • • Stale detection re-enriches after 30 days

Condition Pipeline

  • • Weekly cron checks DOID/HPO/MONDO for updates
  • • Downloads OBO/OWL → parses → stages JSON
  • • Merges across datasets using normalized names
  • • Imports to conditions + graph tables

Admin Operations

  • • POST /api/admin/refresh-drug/:name — mark stale
  • • POST /api/admin/refresh-all — re-enrich all
  • • POST /api/admin/refresh-batch — by name list
  • • Next load: needsFdaEnrichment → fires AI remap
